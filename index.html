☁️ Cloud Sync — GitHub Gist
1
สร้าง Personal Access Token
ไปที่ github.com/settings/tokens/new
ตั้งชื่อ เช่น "Loan Tracker Sync" → เลือก Scope: ✅ gist เท่านั้น → Generate token
⚠️ Copy token ไว้ก่อน จะมองเห็นครั้งเดียว
2
Gist ID (ว่างไว้ = สร้างใหม่อัตโนมัติ)
ถ้ามี Gist อยู่แล้ว ใส่ Gist ID (ตัวเลขหลัง gist.github.com/user/ID)
ถ้าเพิ่งเริ่ม ว่างไว้แล้วกด "ทดสอบ" — ระบบจะสร้าง Gist ใหม่ให้
3
ทดสอบการเชื่อมต่อ
กดปุ่มเพื่อทดสอบ Token และ Gist ID
🔒 Token เก็บใน localStorage ของ Browser นี้เท่านั้น ไม่ถูกส่งไปที่ใดนอกจาก GitHub API
